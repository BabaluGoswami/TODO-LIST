🚀 Smart To-Do List App

A simple, modern, and responsive To-Do List Web Application built using HTML, CSS, and JavaScript.
It helps users manage daily tasks efficiently with features like priority, due date, search, and local storage support.

🌐 Live Demo:
👉 https://babalugoswami.github.io/TODO-LIST/

📌 Features
➕ Add new tasks easily
✔ Mark tasks as completed
✏ Edit existing tasks
❌ Delete individual tasks
🗑 Clear all tasks at once
📅 Add due date for tasks
⚡ Priority levels (High / Medium / Low)
🔍 Real-time search functionality
💾 Data saved in Local Storage (no data loss on refresh)
🌙 Dark Mode support
📊 Live statistics (Total / Completed / Pending)
📱 Fully responsive design (mobile + desktop)
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
Local Storage API
🎯 How It Works
Enter your task in the input box
Select due date and priority
Click Add
Manage tasks (edit / complete / delete)
Search or sort tasks easily
Data stays saved in browser automatically
